# Scrum Poker Layout Mockup

## Page Layout

### Task Banner
- A prominent banner at the top of the page displaying the task being voted on.
- Example: "Task: Implement User Authentication"

### Voting Section
- Below the banner, a grid layout representing votes from multiple participants.
- Each vote is represented as a card with the participant's name and their vote.
- Example:
  - **John**: 5
  - **Jane**: 3
  - **Alex**: ?

### Footer
- A footer section with buttons for actions like "Submit Votes" or "Reset Votes".
